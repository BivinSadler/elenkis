# Elenkis website

This folder contains a simple static site for Elenkis.

The homepage uses the updated headline “Extending human knowledge through conversation,” the supplied Elenkis logo, an expanded oral-exam solution narrative, an emphasized “The Elephant in the Room: Scaling” transition, and a responsive Advisory Council section linked from the main navigation.

LinkedIn profile links are included for the team members whose URLs were supplied.

The Advisory Council cards use compact text layouts with LinkedIn logo links. The product walkthrough uses the current 90-point score and detailed report screenshots.

The homepage hero uses a centered editorial layout with a real Elenkis conversation screen and a compact 90-point result overlay.

The hero identifies the Socratic Method with Socrates, and the scaling challenge headline is intentionally split into “The Elephant in the Room:” and “Scale.”

## Files
- `index.html` — page structure and copy
- `styles.css` — layout, colors, typography, and responsive styling
- `assets/` — logo, SVG illustration, and product/problem images

## How to edit
Open `index.html` in a text editor to change the text. Open `styles.css` to change colors, spacing, or layout.

## How to publish on Cloudflare Pages
Upload the whole folder, or connect a GitHub repo that contains these files. The site does not require a build step.
